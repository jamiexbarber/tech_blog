Inverting Op-Amp

What does it do?

An ideal inverting op-amp takes a DC (or AC) signal and amplifies it based on the gain set by two resistors. The input resistance to the op-amp is set by an input resistor (R1). The output resistance is ideally zero. The gain comes with a 180 degree phase shift.

Can you help me derive the transfer function?

Transfer function is language that comes from analyzing circuits in the frequency domain. We'll discuss this in more detail when we discuss Bode Plots and filters. For now, consider the transfer function as the ratio between the output voltage (Vout) and the input voltage (Vin), or put more simply, the gain.
To derive the transfer function (gain) for the ideal inverting op-amp, we will use Kirchoff's Current Law, which says current flowing into a node must equal current flowing out of a node. Why is the virtual ground a valid assumption?

The virtual ground is the idea that the non-inverting terminal and the inverting terminal have the same voltage potential (or very close). To understand the root of this assumption, we'll introduce another ideal characteristic, very large open-loop gain. Open-loop gain is gain inherent to the op-amp, it's the gain of the op-amp without the feedback resistor (R2) connected. It's usually on the order of 104 or 105.
This type of gain from open-loop would lead to op-amp instability if left unchecked. Designers use negative feedback (connecting the output to the inverting terminal through a feedback resistor (R2)) to stabalize the amplifier and ensure the output voltage is within the available range (as supplied by the + and - power supplies or the rails)
Negative feedback in combination with, near-infinite open-loop gain, helps ensure the inverting terminal stays close in value to the non-inverting terminal. This can proven mathmatically as well with the op-amp equivalent circuit (or the small-signal circuit).

What applications is the inverting op-amp well-suited for?

Inverting op-amps are well-suited for applications where a designer needs control over the input impedance into the op-amp. This may be particularly relevant for noise calculations. Additionally, an inverting op-amp makes sense if a designer needs (or is willing to work with) a 180 degree phase shift in the output.
