Non-Inverting Op-Amp

What does it do?

An ideal non-inverting op-amp takes a DC (or AC) signal and amplifies the signal based on the gain set by two resistors, similar to an inverting op-amp. Unlike the inverting op-amp, the input resistance to the op-amp is set by the input resistance inherent to the op-amp.
The input resistance to the two input terminals to an op-amp is usually set by MOSFET transistors which have a very high input resistance (due to the gate oxide, an insulation layer, between the gate and the channel, there is a further discussion on the MOSFET basics page).
The first resistor (R1) is connected to the inverting terminal to ground and the second resistor (R2) serves as feedback resistor connected to the inverting terminal as well, right before (R1). The input voltage goes into the non-inverting terminal.
The output resistance is ideally zero, and unlike with the gain of the inverting op-amp, there is no phase shift to worry about for the gain.

Can you help me derive the transfer function?

We'll define the transfer function as simply the ratio of the output voltage (Vout) relative to the input voltage (Vin) i.e. the gain. (There will be a more detailed discussion on transfer function on the Bode plot and filters page)
To derive the transfer function (gain) for the ideal non-inverting op-amp, we will use Kirchoff's Current Law, which says the current flowing into a node must equal current flowing out of a node. If the virtual ground didn't exists, what can designers do to help minimize the voltage drop between V+ and V-

The virtual ground is the idea that the non-inverting terminal and the inverting terminal have the same voltage potential (or very close). You can read more about it here. Note that virtual ground is an assumption made for ideal Op-Amps. In reality, the two terminals are not exactly the same and differ from each other by some voltage offset. We will consider input and output voltage offset in a later article.
For now, consider the additional current that the negative terminal would see, just by being connected to two resistors. We can offset that additional current (which leads to the additional voltage potential) by adding a third resistor.

What applications is the non-inverting op-amp well-suited for?

Inverting op-amps are well-suited for applications where a designer needs a really high input impedance, which can particularly helpful when considering voltage buffers. A typical voltage buffer is simply R1 set to open and the feedback resistor set to a short. Voltage buffers help transfer the maximum amount of voltage (and therefore power) on to the next stage in a circuit system. They help do this through a larger input resistance (very little current, so very little voltage drop) and a very small output resistance (as much current as possible transfers over)
